# Earnest Wiki

AI-generated documentation repository for GitHub projects.

## About

This repository hosts comprehensive documentation automatically generated by [Earnest AI Tools](https://earnest-ai-tools.staging.earnest.com/wiki-agent). Each wiki is created by analyzing repository code and generating detailed, searchable documentation.

## Structure

```
earnest-wiki/
  wikis/
    owner-repo/          # Each repository gets its own directory
      .vitepress/
        config.ts        # Wiki-specific configuration
      index.md           # Wiki homepage
      *.md               # Documentation pages
```

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run docs:dev

# Build for production
npm run docs:build

# Preview production build
npm run docs:preview
```

## Publishing Wikis

Wikis are published automatically via pull requests:

1. Generate documentation using Earnest AI Tools Wiki Agent
2. Click "Publish" to create a PR
3. Review and merge the PR
4. GitHub Actions will automatically deploy to GitHub Pages

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions whenever changes are pushed to the main branch.
